CSS - Cascading Style Sheets
O CSS (Cascading Style Sheets), ou Folhas de Estilo em Cascata, é a linguagem responsável por dar aparência às páginas da web. Enquanto o HTML funciona como a estrutura de um site, o CSS atua como a “roupa e decoração”, definindo cores, fontes, espaçamentos, layouts e até animações. Sua importância está na separação entre conteúdo e estilo, permitindo que o HTML se concentre na informação e o CSS no design.
